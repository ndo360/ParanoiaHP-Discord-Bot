# ParanoiaHP-Discord-Bot
This is the source code for my discord bot created for Paranoia high programmers. Enjoy!

How to set up:
1. Install Node.js
2. Create a folder for the bot *essential*
3. Open cmd in adminstrative mode
4. Type "cd desktop"
5. Type "cd <filename>"
6. Type "npm install discord.js"
7. Open up PHP bot in a code editing software and make your adjustments (there are notes)
8. Once finished, launch "autorun"
9. Invite the bot and enjoy!
